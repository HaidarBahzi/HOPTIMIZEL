# HOPTIMIZEL

This Program is like [HOPTIMIZE](https://github.com/haidarbahzi/HOPTIMIZE) but for linux.

# Feature

This program contain some feature such as :
- Delete Temporary Files
- Repair System Files
- Repair Missing Update
- Update Linux
- Show System Info
